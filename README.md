# E-Commerce Sales Analysis using Python

This project performs end-to-end exploratory data analysis (EDA) on an e-commerce retail dataset to identify sales trends, top-performing products, and country-wise revenue insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- The UK dominates total sales revenue
- A small number of products generate most revenue
- Sales show seasonal monthly trends

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the notebook or Python script
